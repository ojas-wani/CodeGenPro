# CodeGenPro
**Use LLMs to improve the code quality and debugging.**  
  **Note:** Project is under development in a private repo. Mail for more detailed information: [ojaswani203@gmail.com](mailto:ojaswani203@gmail.com)  

### Functionality:
* Code Generation based on description
* Code-quality checking using rule-set in Python and Java
* Improve code quality based on reports
### Technologies:
* LLM API (Groq)
* Function Calling
* Multi-Agent
* Multi-Modal
